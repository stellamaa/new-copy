# Images Directory

This directory is for your portfolio images. 

## How to add your images:

1. Add your portfolio images to this `images/` folder
2. Update the image paths in `script.js` in the `config.portfolio` array
3. Replace the placeholder image paths with your actual image filenames

## Recommended image specifications:
- **Format**: JPG or PNG
- **Size**: 800x600 pixels or larger
- **Aspect ratio**: 4:3 or 16:9 works best
- **File size**: Keep under 2MB for fast loading

## Example:
If you add an image called `my-project.jpg`, update the script.js file:
```javascript
{
    title: "My Amazing Project",
    client: "Client Name",
    year: "2024", 
    service: "Web Design",
    image: "images/my-project.jpg", // Your image path
    description: "Description of your project"
}
```

## Placeholder Images
If an image fails to load, a placeholder will be automatically displayed.
